## Tier 1: Primary Platforms (Daily Use)

### 1. LinkedIn
**URL:** https://www.linkedin.com/jobs/

**Why It's #1 for You:**
- Largest professional network with strong semiconductor/EDA presence
- Direct access to recruiters at Synopsys, Cadence, Siemens EDA, Silvaco competitors
- Easy Apply feature for quick applications
- Company follow alerts for target employers

**Recommended Search Queries:**
```
"Staff SDET" remote
"Principal SDET" remote
"EDA Engineer" remote
"Test Infrastructure Architect" remote
"QA Automation Engineer" semiconductor remote
"VLSI Engineer" remote
"CAD Engineer" EDA remote
"Software Engineer in Test" remote
"Senior SDET" remote
"Lead QA Engineer" remote
```

**Profile Optimization:**
- Headline: `Staff/Principal SDET | EDA & VLSI | 18+ Years QA Automation | Test Infrastructure Architect`
- Enable "Open to Work" (visible to recruiters only if preferred)
- Set location preference: `Armenia` with `Remote` selected
- Add skills: Python, Bash, CI/CD, Test Automation, EDA, VLSI, SDET, QA Architecture

**Job Alerts Setup:**
1. Search your target titles with "Remote" filter
2. Click "Create alert"
3. Set frequency: Daily
4. Enable email + push notifications

## Remote Work Verification Checklist

Before applying, verify:
- [ ] Job posting explicitly states "Fully Remote" or "Remote (Country)"
- [ ] Company has remote-first culture (check Glassdoor reviews)
- [ ] Time zone overlap is acceptable (e.g., Armenia to US PST = 10-12 hours)
- [ ] No relocation requirement mentioned
- [ ] Equipment/shipping policy for remote employees
- [ ] Payment method for Armenia (wire transfer, Payoneer, etc.)
